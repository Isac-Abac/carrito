# Carrito de Compras - Productos de Informática

Proyecto web hecho con HTML, CSS y JavaScript puro.

## Funcionalidades incluidas
- Agregar productos al carrito.
- Mostrar total en quetzales (GTQ).
- Quitar productos y vaciar carrito.
- Validar datos de pago.
- Generar recibo en PDF.
- Cambiar entre modo claro y oscuro.
- Vista previa de imágenes de productos.

## Archivos creados
- `index.html`: estructura principal de la interfaz.
- `styles.css`: estilos visuales y diseño responsive.
- `app.js`: lógica del catálogo, carrito, validaciones y PDF.

## Cómo ejecutar
1. Abre `index.html` en tu navegador.
2. Agrega productos al carrito.
3. Completa el formulario de pago para generar el recibo PDF.
